# Changelog

All notable changes to RelayDesk are documented here.

## Unreleased

### Added

- Multi-Agent Hermes connections with per-member access control.
- Conversation history, pinned/archive/delete controls, controlled file storage,
  real Hermes streaming, and browser-visible scheduled task results.
- Optional Content Workspace controlled with
  `RELAYDESK_CONTENT_WORKSPACE_ENABLED`.
- English and Simplified Chinese locale selection foundation.
- Docker Compose deployment, health checks, backups, security and contributor
  guidance.
